# ❤️ Heart Disease Prediction 

## Overview
Heart Disease Prediction is a Machine Learning project that predicts whether a patient is likely to have heart disease based on medical information. The project uses Logistic Regression and a Flask web application for prediction.

## Features
- Heart Disease Prediction
- User-friendly Interface
- Flask Web Application
- Machine Learning Model
- Fast Prediction

## Installation

1. Clone Repository

git clone https://github.com/yourusername/Heart-Disease-Prediction.git

2. Install Libraries

pip install -r requirements.txt

3. Train Model

python model_training.py

4. Run Application

python app.py

5. Open Browser

http://127.0.0.1:5000
